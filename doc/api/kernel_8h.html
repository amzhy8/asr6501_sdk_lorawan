◆ aos_event_get()

int aos_event_get ( aos_event_t *  event,
unsigned int  flags,
unsigned char  opt,
unsigned int *  actl_flags,
unsigned int  timeout 
)

This function will try to get flag set from given event, if the request flag set is satisfied, it will return immediately, if the request flag set is not satisfied with timeout(RHINO_WAIT_FOREVER,0xFFFFFFFF), the caller task will be pended on event until the flag is satisfied, if the request flag is not satisfied with timeout(RHINO_NO_WAIT, 0x0), it will also return immediately. Note, this function should not be called from interrupt context because it has possible to lead context switch and an interrupt has no TCB to save context.

Parameters
[in]eventevent object pointer.
[in]flagsrequest flag set.
[in]optoperation type, such as AND,OR,AND_CLEAR,OR_CLEAR.
[out]actl_flagsthe internal flags value hold by event.
[in]flagsrequest flag set.
[in]timeoutmax wait time in millisecond.
Returns
0: success.
